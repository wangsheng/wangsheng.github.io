Feb 22, 2017 | PHP | Hits

退出

使用退出代码是控制台应用程序开发的最佳实践。通常，一个命令返回0说明一切都OK。如果它是一个大于零的数，可以理解为存在一个错误。返回的数值会是错误代码，它可能在发现错误详情方面会有用。 例如，1 一般可以代表一个未知的错误，用其他大于1的代码来代指一些特定的错误：输入错误，文件丢失等等。
